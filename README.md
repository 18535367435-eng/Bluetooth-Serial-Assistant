# Bluetooth-Serial-Assistant
一款基于蓝牙SPP协议的轻量跨平台无线串口通信工具。 / A lightweight, cross-platform tool for wireless serial communication based on the Bluetooth SPP protocol.

## 概述 / Overview

将手机变为便携式无线串口终端，实现移动设备与串口设备之间的无线数据交互。专为需要快速、无线缆串口通信的开发者、硬件爱好者和工程师设计。 / Turn your mobile phone into a portable wireless serial terminal to realize wireless data interaction between mobile devices and serial devices. Designed for developers, hardware enthusiasts, and engineers who need fast, cable-free serial communication.

## 核心功能 / Core Features

-   扫描并连接周边蓝牙SPP设备 / Scan and connect to nearby Bluetooth SPP devices
    
-   双向数据传输 / Bidirectional data transmission
    
-   可自定义预设指令，方便快速发送 / Customizable preset commands for quick sending
    
-   实时数据接收与日志记录 / Real-time data reception and log recording
    

##  应用场景 / Application Scenarios

-   嵌入式系统调试（单片机、开发板等） / Embedded system debugging (microcontrollers, development boards, etc.)
    
-   蓝牙模块配置与测试 / Bluetooth module configuration and testing
    
-   IoT设备原型开发与数据交互 / IoT device prototyping and data interaction
    
-   DIY硬件项目与智能家居控制 / DIY hardware projects and smart home control
    
-   各类硬件的无线串口通信需求 / Wireless serial communication needs for various hardware
    

##  使用指南 / Usage Guide

1.  打开小程序，开启手机蓝牙 / Open the mini-program and enable Bluetooth on your phone
    
2.  扫描并选择目标蓝牙SPP设备进行连接 / Scan and select the target Bluetooth SPP device to connect
    
3.  发送自定义数据或预设指令，实时查看接收数据 / Send custom data or preset commands, and view received data in real time
    
4.  按需导出日志用于后续分析 / Export logs as needed for subsequent analysis
    

##  优势 / Advantages

-   便携：无需携带电脑或有线适配器 / Portable: No need to carry a computer or wired adapter
    
-   易用：界面直观，学习成本低 / Easy to use: Intuitive interface with low learning cost
    
-   轻量：启动快速，资源占用少 / Lightweight: Fast startup and low resource consumption
    
-   通用：兼容大多数支持蓝牙SPP协议的设备 / Universal: Compatible with most devices supporting Bluetooth SPP protocol
